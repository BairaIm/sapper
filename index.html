Описание

При нажатии на ячейку, будет показано число мин вокруг открытой ячейку, Ваша задача открыть все ячейки без мин. Флажком отмечайте поля где с миной, ячейки с флажком не будут открываться.

Органы управления (компьютерная версия / мобильная версия):
- нажатие левой кнопки мыши по ячейке / нажатие на ячейку - открыть ячейку;
- нажатие правой кнопки мыши по ячейке / нажать на кнопку с одним флажком (для установки одного флажка) или на кнопку с двумя флажками (для установки множества флажков) и затем нажать на ячейку - установить флажок (обозначить место мины и исключить случайное открытие ячейки);
- двойное нажатие левой кнопки мыши по ячейке, нажатие колесом мыши / двойное нажатием на ячейку - открывает все ячейки вокруг (действует только для открытых ячеек, с количеством флажком вокруг, соответствующем числу на ячейке).

Конец игры:
- Победа (смайлик в очках) - открыты все ячейки не содержащие мины;
- Поражение (грустный смайлик) - открыта мина;

Для запуска новой игры нажмите на смайлик.
